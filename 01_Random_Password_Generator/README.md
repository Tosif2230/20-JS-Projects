# Random Password Generator

A simple and responsive **Random Password Generator** built using **HTML, CSS, and JavaScript**.

### Features

* Generate a secure **14-character random password**.
* Password includes:

  * Uppercase letters (A–Z)
  * Lowercase letters (a–z)
  * Numbers (0–9)
  * Special symbols (!@#$%^&* etc.)
* **Copy to Clipboard** functionality with a single click.
* Displays a **toast notification** when the password is copied successfully.
* Toast message automatically hides after **2 seconds**.
* Clean and user-friendly interface.

### Technologies Used

* HTML5
* CSS3 (Vanilla CSS)
* JavaScript (Vanilla JS)

### How It Works

1. Click the **Generate Password** button to create a new 14-character password.
2. Click the **Copy** button to copy the generated password to your clipboard.
3. A toast notification appears confirming the password has been copied and disappears automatically after 2 seconds.

This project is ideal for learning JavaScript fundamentals such as random string generation, DOM manipulation, clipboard API usage, and toast notifications.
